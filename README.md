## 📖 Glossary

- **WCAG** – Web Content Accessibility Guidelines: international standards for web accessibility  
- **RGAA** – Référentiel Général d’Amélioration de l’Accessibilité: French accessibility framework  
- **RAAM** – Référentiel d’Accessibilité des Applications et des Médias: accessibility framework for applications and media  
- **ARIA** – Accessible Rich Internet Applications: HTML attributes that improve accessibility for dynamic content  
- **NVDA / JAWS / VoiceOver / TalkBack** – Popular screen readers for testing accessibility  
- **UI / UX** – User Interface / User Experience  
- **CRUD** – Create, Read, Update, Delete (basic operations in apps)
